# theme/ フォルダについて

このフォルダには、2BP の **テーマ（配色）・アイコン・通知音** などの見た目に関するファイルを置きます。
ここでは「テーマの仕組み」と「自分でテーマを作る／カスタマイズする方法」を説明します。

## フォルダ構成

```
theme/
├── README.md          ← このファイル
├── icon.ico            ← (共通) アプリアイコン。各テーマ側に無い場合はこちらが使われる
├── icon.png             (icoが無い場合はpngでも可)
├── ng_se.wav            ← (共通) NG通知の効果音。各テーマ側に無い場合はこちらが使われる
│
├── dark/
│   ├── theme.json       ← このテーマの配色定義（必須）
│   ├── icon.ico          (省略可: このテーマ専用アイコン)
│   └── ng_se.wav         (省略可: このテーマ専用の通知音)
│
├── light/
│   └── theme.json
│
├── dark_pink/
│   └── theme.json
│
└── light_pink/
    └── theme.json
```

- 各テーマは `theme/<テーマ名>/` という**サブフォルダ**として存在し、その中に必ず `theme.json` を置きます。
- `theme.json` が無いサブフォルダはテーマ一覧に表示されません。
- `icon.ico` / `icon.png` / `ng_se.wav` は、**各テーマフォルダの中に置けば、そのテーマだけの専用ファイルとして優先的に使われます**。テーマフォルダの中に無い場合は、`theme/` フォルダ直下（共通）のファイルが使われます。

---

## テーマの選び方・追加方法

- 2BP の設定画面でテーマ（`dark` / `light` / `dark_pink` / `light_pink` など）を選択できます。
- テーマの一覧は、`theme/` フォルダ内をスキャンして「`theme.json` を持つフォルダ」を自動で探して作られます。
- そのため、**新しいテーマを追加したい場合は、`theme/` の下に新しいフォルダを作り、その中に `theme.json` を置くだけで一覧に表示されるようになります**。

## `theme.json` の書き方

`theme.json` は以下のような構造の JSON ファイルです。大きく **`ui`**（アプリ全体のウィンドウ・ボタンなどの配色）と **`thread`**（スレッド/カタログのHTML表示部分の配色）の2セクションに分かれています。

```json
{
  "ui": {
    "window_bg": "#1e1e1e",
    "text_primary": "#e8e8e8",
    "btn_bg": "#3a3a3a"
  },
  "thread": {
    "body_bg": "#FFFFEE",
    "body_fg": "#7B0004",
    "reply_bg": "#F0E0D6"
  }
}
```

**ポイント:**

- 値はすべて `"#RRGGBB"` 形式の16進数カラーコードです。
- **すべてのキーを書く必要はありません。** 書かれていないキーは、下表の「デフォルト値」が自動的に使われます。
  そのため、ベースのテーマ（例: `dark/theme.json`）をコピーして、変えたい色だけ書き換えるのが簡単です。
- キーを書き間違えても起動エラーにはなりません（単にデフォルト値が使われるだけです）。

## アイコン・通知音について

| ファイル | 説明 |
|---|---|
| `icon.ico` / `icon.png` | アプリのウィンドウアイコン・タスクバーアイコンです。`.ico` を優先して読み込みます。テーマフォルダ内に置けばそのテーマ専用、`theme/` 直下に置けば全テーマ共通のアイコンになります。 |
| `ng_se.wav` | NGワード・NG画像・逆NGなどの設定に一致したレスを検知した際に再生される通知音（`.wav`）です。テーマフォルダ内に置けばそのテーマ専用、`theme/` 直下に置けば全テーマ共通の通知音になります。差し替える場合は同じファイル名のまま置き換えてください。 |

> 同梱の `ng_se.wav` の音声生成には **Irodori-TTS**（Aratako 氏作・MIT License）を利用しています。詳細はリポジトリ直下の [README.md](../README.md) の「使用している音声について」を参照してください。

## 独自テーマを作る手順

1. `theme/` フォルダ内の `dark/`（または近い配色のテーマ）フォルダを丸ごとコピーします。
2. コピーしたフォルダの名前を、好きなテーマ名（例: `mytheme`）に変更します。
   - フォルダ名は半角英数字推奨です。この名前が設定画面のテーマ選択肢に表示されます。
3. `mytheme/theme.json` を開き、変更したい色だけを書き換えます。
   - 上記の「`ui`」「`thread`」セクションの表を参考に、変えたいキーの値（`"#RRGGBB"`）を書き換えてください。
   - 書かないキーは元のテーマと同じデフォルト値が使われます。
4. （任意）`mytheme/icon.ico` や `mytheme/ng_se.wav` を置けば、そのテーマ専用のアイコン・通知音にできます。
5. 2BP を再起動（またはテーマ一覧を再読み込み）すると、設定画面のテーマ選択に `mytheme` が表示されます。
